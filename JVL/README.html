JVL (Java Vortex Lattice)

Lockheed Martin Proprietary Tool

Description

A Java based vortex lattice code. Vortex Lattice is a potential flow based approach that is useful to quickly generate aerodynamic characteristics of a vehicle. Particularly useful to extend into unsteady effects for aeroelastic behavior.
